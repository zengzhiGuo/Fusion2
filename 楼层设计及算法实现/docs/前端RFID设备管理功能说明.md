# 前端RFID设备管理功能说明

## 功能概述
在地图上通过鼠标点击标注RFID设备的位置，并可以关联到楼梯或房间。RFID设备标记只会显示在其所在楼层的地图上。

## 使用步骤

### 1. 选择楼层
在左侧楼层列表中选择要添加RFID设备的楼层。

### 2. 点击添加RFID设备按钮
点击地图预览区域右上角的 📡 按钮，进入RFID设备标注模式。

### 3. 在地图上标注位置
- 系统会提示"请在地图上点击选择RFID设备的位置"
- 鼠标光标会变成十字准星样式
- 在地图上点击要标注RFID设备的位置
- 点击后会自动打开RFID设备信息表单

### 4. 填写RFID设备信息
表单会自动填充以下信息：
- **设备坐标**：显示刚才点击的位置坐标（X, Y，单位：米）
- **设备名称**：输入RFID设备名称（必填）
- **所在楼层**：自动设置为当前楼层ID（只读，必填）
- **所属楼梯**：从下拉列表中选择楼梯（可选，默认为空）
- **所属房间/门**：从下拉列表中选择房间（可选，默认为空）
- **描述**：可选的描述信息

**重要说明：**
- 所属楼梯和所属房间都是可选的，可以都不选
- 下拉列表只显示当前楼层的楼梯和房间
- 可以只选择楼梯，或只选择房间，或两者都不选

### 5. 提交保存
点击"确认添加"按钮，RFID设备信息会保存到数据库，并在地图上显示为橙色圆点。

## 地图显示
- 已标注的RFID设备会在**其所在楼层（map_id）**的地图上显示为**橙色圆点**
- RFID设备标记不会显示在其他楼层的地图上
- 鼠标悬停在RFID设备标记上会高亮显示
- 设备名称会显示在标记上方

## RFID设备管理功能

### 查看RFID设备详情
点击地图上的橙色RFID设备标记，会弹出设备详情窗口，显示：
- 设备名称
- 坐标位置
- 所在楼层
- 所属楼梯（如果有）
- 所属房间（如果有）
- 描述信息（如果有）

### 编辑RFID设备
在RFID设备详情窗口中点击"✏️ 编辑"按钮：
- 打开编辑表单，可以修改设备名称、所属楼梯、所属房间和描述
- 所在楼层和坐标不可修改
- 点击"保存修改"提交更改

### 删除RFID设备
删除RFID设备有两种方式：
1. 在RFID设备详情窗口中点击"🗑️ 删除"按钮
2. 在编辑表单中点击"删除"按钮

删除前会弹出确认对话框，确认后RFID设备标记会从地图上移除。

## 操作流程总结

1. **添加RFID设备**：选择楼层 → 点击📡按钮 → 在地图上点击位置 → 填写设备信息 → 选择所属楼梯/房间（可选）→ 确认添加
2. **查看RFID设备**：点击地图上的橙色RFID设备标记
3. **编辑RFID设备**：查看设备详情 → 点击"编辑"按钮 → 修改信息 → 保存修改
4. **删除RFID设备**：查看设备详情 → 点击"删除"按钮 → 确认删除

## 与楼梯和门的对比

| 特性 | 楼梯 | 门 | RFID设备 |
|------|------|-----|----------|
| 标记颜色 | 绿色 | 蓝色 | 橙色 |
| 标记形状 | 圆形 | 矩形 | 圆形（较小） |
| 主要信息 | 楼梯名称、上下层连接 | 房间名称 | 设备名称、所属楼梯/房间 |
| 特殊功能 | 可以跳转到上下层 | 无 | 可关联楼梯或房间 |
| 按钮图标 | 🪜 | 🚪 | 📡 |
| 按钮悬停色 | 绿色 | 蓝色 | 橙色 |

## 数据库字段
RFID设备信息保存在 `ship_floor.rfid_devices` 表中：
- `id`: RFID设备的唯一标识（自增主键）
- `device_name`: 设备名称（必填）
- `map_id`: RFID设备所在楼层ID（必填）
- `x`: RFID设备X坐标（米，必填）
- `y`: RFID设备Y坐标（米，必填）
- `stairway_id`: 所属楼梯ID（可选）
- `door_id`: 所属门ID（可选）
- `description`: 描述信息（可选）
- `is_active`: 是否启用
- `create_time`: 创建时间
- `update_time`: 更新时间

## 注意事项
1. 必须先选择楼层才能添加RFID设备
2. 必须在地图上点击选择位置后才能提交
3. 设备名称和所在楼层为必填项
4. 所属楼梯和所属房间都是可选的，可以都不选
5. 下拉列表只显示当前楼层的楼梯和房间
6. RFID设备标记只显示在 map_id 对应的地图上
7. 坐标系统基于地图的物理尺寸（米）
8. 删除操作是软删除，数据不会真正从数据库中移除
9. 如果删除了门，关联的RFID设备的 door_id 会自动设为 NULL（外键约束）

## 使用场景
- 在楼梯处安装RFID设备：选择所属楼梯
- 在房间门口安装RFID设备：选择所属房间
- 在走廊等公共区域安装RFID设备：两者都不选
