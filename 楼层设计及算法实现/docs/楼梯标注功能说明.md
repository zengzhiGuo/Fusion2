# 楼梯标注功能说明

## 功能概述
在地图上通过鼠标点击标注楼梯位置，并保存楼梯的连接信息。楼梯标记只会显示在其所在楼层的地图上。

## 使用步骤

### 1. 选择楼层
在左侧楼层列表中选择要添加楼梯的楼层。

### 2. 点击添加楼梯按钮
点击地图预览区域右上角的 ➕ 按钮，进入楼梯标注模式。

### 3. 在地图上标注位置
- 系统会提示"请在地图上点击选择楼梯位置"
- 鼠标光标会变成十字准星样式
- 在地图上点击要标注楼梯的位置
- 点击后会自动打开楼梯信息表单

### 4. 填写楼梯信息
表单会自动填充以下信息：
- **楼梯坐标**：显示刚才点击的位置坐标（X, Y，单位：米）
- **楼梯名称**：默认为"[当前楼层名称] 楼梯"，可修改（必填）
- **所在楼层**：自动设置为当前楼层ID（只读，必填）
- **上层楼层**：默认为当前楼层ID，可从下拉列表中修改或留空（可选）
- **下层楼层**：从下拉列表中选择连接的下层楼层或留空（可选）
- **描述**：可选的描述信息

注意：
- 所在楼层决定了楼梯标记显示在哪个地图上
- 上层和下层楼层表示逻辑连接关系，都是可选的

### 5. 提交保存
点击"确认添加"按钮，楼梯信息会保存到数据库，并在地图上显示为绿色圆点。

## 地图显示
- 已标注的楼梯会在**其所在楼层（map_id）**的地图上显示为**绿色圆点**
- 楼梯标记不会显示在 upper_map_id 或 lower_map_id 对应的地图上
- 鼠标悬停在楼梯标记上会高亮显示
- 楼梯名称会显示在标记上方

## 数据库字段
楼梯信息保存在 `ship_floor.stairways` 表中：
- `stairway_id`: 楼梯唯一标识（自动生成）
- `stairway_name`: 楼梯名称
- `map_id`: 楼梯所在楼层ID（决定标记显示在哪个地图上，必填）
- `upper_map_id`: 上层楼层ID（逻辑连接关系，可选）
- `lower_map_id`: 下层楼层ID（逻辑连接关系，可选）
- `x`: 楼梯X坐标（米）
- `y`: 楼梯Y坐标（米）
- `description`: 描述信息
- `is_active`: 是否启用
- `create_time`: 创建时间
- `update_time`: 更新时间

## 注意事项
1. 必须先选择楼层才能添加楼梯
2. 必须在地图上点击选择位置后才能提交
3. 楼梯名称和所在楼层为必填项
4. 上层和下层楼层都是可选的，可以留空
5. 楼梯标记只显示在 map_id 对应的地图上
6. 坐标系统基于地图的物理尺寸（米）


## 楼梯管理功能

### 查看楼梯详情
点击地图上的绿色楼梯标记，会弹出楼梯详情窗口，显示：
- 楼梯名称
- 坐标位置
- 所在楼层
- 上层楼层（如果有）
- 下层楼层（如果有）
- 描述信息（如果有）

### 楼层跳转
在楼梯详情窗口中：
- 如果楼梯有 `upper_map_id`（上层楼层），会显示"⬆️ 前往上层"按钮
- 如果楼梯有 `lower_map_id`（下层楼层），会显示"⬇️ 前往下层"按钮
- 点击按钮可以快速跳转到对应楼层的地图

### 编辑楼梯
在楼梯详情窗口中点击"✏️ 编辑"按钮：
- 打开编辑表单，可以修改楼梯名称、上下层连接关系和描述
- 所在楼层和坐标不可修改
- 点击"保存修改"提交更改

### 删除楼梯
删除楼梯有两种方式：
1. 在楼梯详情窗口中点击"🗑️ 删除"按钮
2. 在编辑表单中点击"删除"按钮

删除前会弹出确认对话框，确认后楼梯标记会从地图上移除。

## 操作流程总结

1. **添加楼梯**：选择楼层 → 点击➕按钮 → 在地图上点击位置 → 填写信息 → 确认添加
2. **查看楼梯**：点击地图上的绿色楼梯标记
3. **跳转楼层**：查看楼梯详情 → 点击"前往上层"或"前往下层"按钮
4. **编辑楼梯**：查看楼梯详情 → 点击"编辑"按钮 → 修改信息 → 保存修改
5. **删除楼梯**：查看楼梯详情 → 点击"删除"按钮 → 确认删除
