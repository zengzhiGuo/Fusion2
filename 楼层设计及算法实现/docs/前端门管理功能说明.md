# 前端门管理功能说明

## 功能概述
在地图上通过鼠标点击标注门的位置，并保存门的房间信息。门标记只会显示在其所在楼层的地图上。

## 使用步骤

### 1. 选择楼层
在左侧楼层列表中选择要添加门的楼层。

### 2. 点击添加门按钮
点击地图预览区域右上角的 🚪 按钮，进入门标注模式。

### 3. 在地图上标注位置
- 系统会提示"请在地图上点击选择门的位置"
- 鼠标光标会变成十字准星样式
- 在地图上点击要标注门的位置
- 点击后会自动打开门信息表单

### 4. 填写门信息
表单会自动填充以下信息：
- **门坐标**：显示刚才点击的位置坐标（X, Y，单位：米）
- **房间名称**：输入房间名称（必填）
- **所在楼层**：自动设置为当前楼层ID（只读，必填）
- **描述**：可选的描述信息

### 5. 提交保存
点击"确认添加"按钮，门信息会保存到数据库，并在地图上显示为蓝色方块。

## 地图显示
- 已标注的门会在**其所在楼层（map_id）**的地图上显示为**蓝色方块**
- 门标记不会显示在其他楼层的地图上
- 鼠标悬停在门标记上会高亮显示
- 房间名称会显示在标记上方

## 门管理功能

### 查看门详情
点击地图上的蓝色门标记，会弹出门详情窗口，显示：
- 房间名称
- 坐标位置
- 所在楼层
- 描述信息（如果有）

### 编辑门
在门详情窗口中点击"✏️ 编辑"按钮：
- 打开编辑表单，可以修改房间名称和描述
- 所在楼层和坐标不可修改
- 点击"保存修改"提交更改

### 删除门
删除门有两种方式：
1. 在门详情窗口中点击"🗑️ 删除"按钮
2. 在编辑表单中点击"删除"按钮

删除前会弹出确认对话框，确认后门标记会从地图上移除。

## 操作流程总结

1. **添加门**：选择楼层 → 点击🚪按钮 → 在地图上点击位置 → 填写房间名称 → 确认添加
2. **查看门**：点击地图上的蓝色门标记
3. **编辑门**：查看门详情 → 点击"编辑"按钮 → 修改信息 → 保存修改
4. **删除门**：查看门详情 → 点击"删除"按钮 → 确认删除

## 与楼梯的区别

| 特性 | 楼梯 | 门 |
|------|------|-----|
| 标记颜色 | 绿色圆点 | 蓝色方块 |
| 标记形状 | 圆形 | 矩形 |
| 主要信息 | 楼梯名称、上下层连接 | 房间名称 |
| 特殊功能 | 可以跳转到上下层 | 无跳转功能 |
| 按钮图标 | 🪜 | 🚪 |
| 按钮悬停色 | 绿色 | 蓝色 |

## 数据库字段
门信息保存在 `ship_floor.doors` 表中：
- `id`: 门的唯一标识（自增主键）
- `room_name`: 房间名称（必填）
- `map_id`: 门所在楼层ID（必填）
- `x`: 门X坐标（米，必填）
- `y`: 门Y坐标（米，必填）
- `description`: 描述信息（可选）
- `is_active`: 是否启用
- `create_time`: 创建时间
- `update_time`: 更新时间

## 注意事项
1. 必须先选择楼层才能添加门
2. 必须在地图上点击选择位置后才能提交
3. 房间名称和所在楼层为必填项
4. 门标记只显示在 map_id 对应的地图上
5. 坐标系统基于地图的物理尺寸（米）
6. 删除操作是软删除，数据不会真正从数据库中移除
