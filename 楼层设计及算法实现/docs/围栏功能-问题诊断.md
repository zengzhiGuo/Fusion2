## 解决方案

### 方案1：使用批处理脚本（最简单）

直接双击运行项目根目录下的 `重启前端服务.bat` 文件。

### 方案2：手动重启（推荐）

#### 步骤1：停止前端服务

**方法A：如果能找到运行前端的终端窗口**
1. 切换到运行 `npm run dev` 的终端窗口
2. 按 `Ctrl + C` 停止服务
3. 如果提示"终止批处理操作吗？"，输入 `Y` 并回车

**方法B：如果找不到终端窗口**
1. 打开任务管理器（Ctrl + Shift + Esc）
2. 找到所有 `Node.js` 进程
3. 右键点击 -> 结束任务

**方法C：使用PowerShell命令**
```powershell
# 停止所有Node进程
Get-Process | Where-Object {$_.ProcessName -eq "node"} | Stop-Process -Force
```

#### 步骤2：重新启动前端服务

```bash
# 进入前端目录
cd 20260207/frontend

# 启动开发服务器
npm run dev
```

#### 步骤3：清除浏览器缓存

1. 打开浏览器开发者工具（F12）
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

或者：
- Chrome/Edge: `Ctrl + Shift + Delete` -> 清除缓存
- 然后按 `Ctrl + F5` 强制刷新

#### 步骤4：验证功能

1. 访问 `http://localhost:5173`（或终端显示的端口）
2. 点击左侧导航 **"系统管理"**
3. 应该能看到以下标签页：
   - 人员管理
   - UWB基站
   - RFID设备
   - 地图管理
   - **电子围栏** ← 新增
   - **围栏白名单** ← 新增
   - 系统设置

## 常见问题排查

### 问题1：重启后仍然看不到

**可能原因**：
- 浏览器缓存未清除
- 前端服务未正确启动
- 端口被占用

**解决方法**：
```bash
# 1. 完全停止Node进程
taskkill /F /IM node.exe

# 2. 检查端口占用（5173是默认端口）
netstat -ano | findstr :5173

# 3. 如果端口被占用，结束占用进程
taskkill /F /PID [进程ID]

# 4. 重新启动
cd 20260207/frontend
npm run dev

# 5. 清除浏览器缓存并强制刷新（Ctrl+Shift+Delete，然后Ctrl+F5）
```

### 问题2：点击"电子围栏"标签页显示空白

**可能原因**：
- 后端服务未启动
- 数据库表未创建
- API请求失败

**解决方法**：
```bash
# 1. 检查后端服务是否运行
# 访问 http://localhost:8080/api/manage/regions
# 应该返回JSON数据（可能是空数组[]）

# 2. 检查数据库表
# 在MySQL中执行：
SHOW TABLES LIKE 'regions';
SHOW TABLES LIKE 'region_coordinates';

# 3. 如果表不存在，执行建表脚本
mysql -u root -p your_database < docs/数据库变更-创建围栏表.sql
```

### 问题3：控制台显示404错误

**错误信息**：
```
GET http://localhost:5173/api/manage/regions 404 (Not Found)
```

**可能原因**：后端服务未启动或端口不是8080

**解决方法**：
```bash
# 1. 启动后端服务
cd 20260207/backend
mvn spring-boot:run

# 2. 检查后端端口配置
# 查看 backend/src/main/resources/application.properties
# 确认 server.port=8080

# 3. 检查前端代理配置
# 查看 frontend/vite.config.ts
# 确认代理配置正确
```

### 问题4：组件导入错误

**错误信息**：
```
Failed to resolve component: FenceTable
```

**可能原因**：组件文件路径错误或文件不存在

**解决方法**：
```bash
# 1. 确认组件文件存在
dir 20260207\frontend\src\components\FenceTable.vue

# 2. 检查ManagePage.vue中的导入语句
# 应该是：
import FenceTable from '../components/FenceTable.vue'

# 3. 如果文件不存在，说明组件未创建，需要创建组件文件
```
